Python学习笔记（1）——简单的Python程序

发表于 2016-11-05   |   分类于 Python   |  

今天开始，就要进入到Python的学习之中啦！

本文仅供个人记录和复习，不用于其他用途

Python有什么用

Python有很多用处，网络应用，包括网站、后台服务等等都有用到Python，还有就是各种小工具以及脚本也需要这门语言。Python是一门解释型语言，这也就意味着它的代码很简洁。当然，Python也可以实现很复杂的程序，不过我们在编写代码的时候尽量做到简洁、优雅。

这里推荐一位大神：http://www.liaoxuefeng.com/

Python的简单使用

Python可以在官方网站进行下载，我用的版本是Python2.7，安装完之后，添加系统变量，在path中加入Python的安装路径。打开命令行，输入python，会显示Python版本，并且出现>>>，没有出现的说明你没有把Python加入到路径。

我们先来写一点简单的代码：

>>> 100 + 200
300

或者你可以这么写：

>>> print '300'
300

这里要说明一下，单引号和双引号在这里都可以，但是不能够混用。

命令行和Python交互模式

我们输入100 + 200就会直接得到结果，是不是很奇怪？不用着急，我们接着往下看

命令行模式

命令行模式就是cmd窗口，我们可以执行python进入Python交互模式，也可以执行python hello.py来运行一个.py文件。

Python交互模式

我们看到>>>就是进入到了Python交互模式，这种模式下只能输入代码并且立即执行，也就是我们上面看到的那种情况。如果我们把100 + 200写在文件中，那么执行这个文件是看不到结果的。我们只能够加上print函数才行。

总而言之，Python交互模式下，输入代码就能立即得到结果。

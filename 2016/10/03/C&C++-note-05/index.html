什么是Lambda表达式

C++11中新增了Lambda表达式，用于定义并创建匿名函数对象，简化编程工作。Lambda表达式就相当于包含共有operator()的匿名结构（或类）。

那么类似于如下表达式：

[](int &Element){cout << element << ' ';}

注意事项

  • 没有使用mutable关键字时，不能够修改捕获列表中指定的状态变量。
  • 编写Lambda表达式时，不要忘记使用const对参数进行限定。
  • 包含多条语句时，必须指定返回类型
  • 不要用很长的Lambda表达式，应该使用函数对象
